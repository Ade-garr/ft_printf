### Welcome to my printf repository 👋

⭐️ The goal of this subject is to recode printf.

## 💡 Usage

- ```make``` => will create the library libftprintf.a that you can include to call ft_printf function
- support :
	- converions : "%c %s %p %d %i %u %x %X %% %n" ✅
	- flags : "-0.*+" ✅
	- précision ✅
	- largeur de champ ✅
